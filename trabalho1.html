Conceito de HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.Tecnologia é fruto da junção entre os padrões HyTime e SGML.

Estrura Basica do HTML

< html>

< head>

< title> Titulo < /title>

< /head>

< body>

< /body>

< /html>

Conceito de CSS

O Cascading Style Sheets (CSS) é uma "folha de estilo" composta por “camadas” e utilizada para definir a apresentação (aparência) em páginas da internet que adotam para o seu desenvolvimento linguagens de marcação (como XML, HTML e XHTML). O CSS define como serão exibidos os elementos contidos no código de uma página da internet e sua maior vantagem é efetuar a separação entre o formato e o conteúdo de um documento.

Estrutura básica do CSS:

p {color: blue;}

Arquivo salvo como style.css,

Estrutura do HTML inserido o código CSS:

< html>

< head>

< link href=”style.css” rel=”stylesheet” type=”text/css”>

< title> Titulo < /title>

< /head>

< body>

< p>Teste de cor azul no paragrafo < /p>

< /body>

< /html>

Diferença do HTML para o CSS

HTML é encarregado de dar a estrutura ao conteúdo, o CSS faz a personalização desse mesma estrutura.Ou seja, o CSS dá o estilo para a página HTML.

Algumas das propriedades e seletores do CSS:

font-family:A propriedade fonte family é usada para definir uma lista de fontes e sua prioridade para apresentação de um elemento em uma página. Se por acaso a primeira fonte da lista não estiver instalada na máquina do usuário ,a segunda será usada e assim por diante até uma fonte ser encontrada e instalada.A font-family possui alguns tipos de familia de fontes, os mais conhecidos são:Serif(Time News Roman,Garamond,Georgia,…);Sans serif(Trebuchet,Arial,Verdana, …) e monospace(Courier,Courier new,Andele Mono…) font-size:A propriedade fonte size permite alterar o tamanho da fonte. font-weight: A font-weight define o negrito ou o “peso” da fonte.Uma fonte pode ser normal ou bold. font-style:font-style altera o estilo ou inclinação do texto.

font-variant:É usada para escolher se a variante irá ser normal ou small-caps.Uma fonte small-caps usa as letras maiusculas com tamanhos reduzidos.figure006 Estilos de fontes variantes. text-transform:Controla a capitalização de um texto.Pode ser escolhido a capitalize, uppercase e lowercase. capitalize:Capitaliza a primeira letra de cada palavra. uppercase:Converte todas as letras para maiuscula. lowercase:Converte todas as letras para minuscula.

text align:Serve para fazer o alinhamento de um texto.O alinhamento pode ter 4 atríbutos: right(direita); left(esquerda); centred(centrado); ou justify(justificado). text-indent: Permite que seja aplicado um recuo à primeira linha de um parágrafo. letter-spacing:Controla os espaçamentos entre os caracteres de um texto. line-height:Serve para especificar a altura de uma linha.

border(top,right,bottom e left)style:Define o estilo da borda.existem várias maneiras de se definir ou modificar uma borda,as mais conhecida são a top,right,bottom e left, respectivamente superior, direita, inferior e esquerda. margin:A margin é a distancia entre os lados de elementos vizinhos.Pode controla-la colocando um numero de px para ela. padding(top, right, bottom e left):Também é conhecido por “enchimento”,ele define a distância entre a borda e o conteúdo do elemento.Você pode controlar o padding indicando um número de px em cada um dos espaços.o padding é dividido por top,right,bottom e left,que significa,respectivamente superior, direita, inferior e esquerda. border style:Há vários tipos de bordas disponíveis para escolha.Veja algumas:figure014 imagens de bordas e seus tipos. border width:Define a espessura das bordas.Pode definir os valores thin,medium e thick (fina,média e grossa), ou um valor numérico equivalente em pixels.figure012 espessuras de bordas. color:Define a cor do primeiro plano de um elemento.

background-color:Define a cor de fundo de algum elemento. seletores id e class:Esses seletores são usados para identificar algo em uma página,ou seja,dar um destaque na página.Muitas pessoas não sabem exatamente qual é a diferença entre o id e a class.A diferença básica é a seguinte: seletor id:O seletor id serve como um identificador único,ou seja,no HTML só poderá ter uma class id é usado principalmente nas criações de menus e em javascript.Para se usar o id em um arquivo css,é necessário que seja colocado um “#” na frente do nome. seletor class:O seletor class pode ser usado quantas vezes a pessoa quiser,permitindo ser atribuido um layout a muitas partes.

Para se criar um class no css,é necessário que seja colocado um “.” na frente do nome. first-line: é usado para adicionar um modelo para a primeira linha do selector especificado.
